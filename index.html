Fastest ORM tool for .Net / .Net.Core

is a simple, fast and lightweight embedded .NET ORM tool



  Install from Nuget

One code for all databases. Besides FREE for everyone - including commercial use

No Dependencies

OrmDb is standalone a single DLL (less than 200kb) fully written in .NET managed code (compatible with .NET 4.x, NETCore 2.2 and above).


You can install via NuGet or just copy DLL to your bin project folder.

Fast and lightweight

OrmDb is a simple and fast ORM solution. Ideal for any applications.


You can run HUGE queries and read data Fast and low resource consumption.

And much more...

  • Many database support such as MsSQL, PostgreSQL, MySQL, MariaDB and Oracle (for now)
  • DDL(Data Definition Language) support
  • DML(Data Manipulation Language) support
  • TCL(transaction Control Language) support
  • Map your POCO class for queries
  • Cross column references (ColumnRef)
  • Aggregate support such as TRIM, SUM, AVG, CONCATE, MAX, CONTAINS and many more
  • DataTable and CRUD operation support
  • DataReader support
No STRING Statements
OrmDb does NOT executes the queries as a string statement.



OrmDb executes the queries as a part of the code like LINQ.

ONE Statement for All Databases
You dont need the multiple queries for different databases with OrmDb. ONE Query can be executed on different connection(s).
