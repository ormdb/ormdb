<k>DropIndex</k> is used to drop an existing index of a table in a database.

<br/>

<st>Definition</st>
<code-tab>
	<code lang="c#">
		namespace Connection {
			public class DDL {

				public string DropIndex(Table.Base Table,
										Table.IndexInfo IndexInfo,
										bool OnlyGenerateScript = false){}

				public string DropIndex(string TableName,
										Table.IndexInfo IndexInfo,
										bool OnlyGenerateScript = false){}

				public string DropIndex(string TableName,
										string IndexName,
										bool OnlyGenerateScript = false){}
	</code>
	<code lang="vb">
		NameSpace Connection
			Public Class DDL

				Public Function DropIndex(Table As Table.Base,
										  IndexInfo As Table.IndexInfo,
										  OnlyGenerateScript As Boolean = False) As String

				Public Function DropIndex(TableName As String,
										  IndexInfo As Table.IndexInfo,
										  OnlyGenerateScript As Boolean = False) As String

				Public Function DropIndex(TableName As String,
										  IndexName As String,
										  OnlyGenerateScript As Boolean = False) As String

	</code>
</code-tab>

<br/>

<st>Examples</st>

<ew/>

<br/>

Following section demonstrates how to drop index of a table.

<code-tab>
	<code lang="c#">
		// by table instance
		cn.DDL.DropIndex(h, h.Indexes.First());

		// by table name
		cn.DDL.DropIndex("table1", "index_name");
	</code>
	<code lang="vb">
		' by table instance
		cn.DDL.DropIndex(h, h.Indexes.First())

		' by table name
		cn.DDL.DropIndex("table1", "index_name")
	</code>
</code-tab>

<br/>

Following section demonstrates how to generate <k>DROP INDEX</k> script.

<code-tab>
	<code lang="c#">
		var script = cn.DDL.DropIndex("table1", "index_name", true);
	</code>
	<code lang="vb">
		Dim script = cn.DDL.DropIndex("table1", "index_name", True)
	</code>
</code-tab>
