<k>TRUNCATE</k> method is used to delete existing records in a table like <k>DELETE</k>. This process also executes the <k>UPDATE STATISTICS</k> for tables, after <b>DELETE</b> operation.

<co type="light">
	<small><b>Namespace : </b>Odb.Connection.DDL</small>
</co>

		public List<string> Truncate(IEnumerable<string> TableNames,

		public string Truncate(string TableName,

<st>Examples</st>
<ew/>
<br/>
<br/>

Following section demonstrates <k>TRUNCATE</k> the table(s).
<code-tab>
	<code lang="c#">
		// single table by instance
		cn.DDL.Truncate(h);

		// multi table by instance
		cn.DDL.Truncate({h, l});

		// single table by name
		cn.DDL.Truncate("table1");
	</code>
	<code lang="vb">
		' single table by instance
		cn.DDL.Truncate(h)

		' multi table by instance
		cn.DDL.Truncate({h, l})

		' single table by name
		cn.DDL.Truncate("table1")
	</code>
</code-tab>
