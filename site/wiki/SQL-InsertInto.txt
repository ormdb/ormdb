<k>InserInto</k> function copies data from one table and inserts into same or another table.


<st>Definition</st>
<code-tab>
	<code lang="c#">
		namespace SQL
			public class X

				InsertInto InsertInto(Table.Base Table, SQL.Select From){}

				InsertInto InsertInto(SQL.TempTable Table, SQL.Select From){}
	</code>
	<code lang="vb">
		NameSpace SQL
			Public Class X

				InsertInto(Table As Table.Base, From As SQL.Select) As SQL.InsertInto

				InsertInto(Table As SQL.TempTable, From As SQL.Select) As SQL.InsertInto
	</code>
</code-tab>


<st>Remarks</st>
Make sure the order of the values is in the same order as the columns in the target table.

<st>Examples</st>
<ew/>
<br/>
<br/>

Following section demonstrates <b>regular insertion from select</b>.
<code-tab>
	<code lang="c#">
		// copies records in 2017 into same table as 2018

		var sql = SQL.X.Select(h.Const(2018).AS(h.Year), 
							   h.Id,
							   h.Note,
							   h.DocDate).From(h, h.Year == 2017);

		SQL.X.InsertInto(h, sql).Execute(cn);
	</code>
	<code lang="vb">
		' copies records in 2017 into same table as 2018

		Dim sql = SQL.X.Select(h.Const(2018).AS(h.Year), 
							   h.Id,
							   h.Note,
							   h.DocDate).From(h, h.Year = 2017)

		SQL.X.InsertInto(h, sql).Execute(cn)
	</code>
</code-tab>
