<k>ExecuteReader</k> method is used to execute a query as <b>FORWARD_ONLY</b>.
<co type="warning">
	These methods are come as an extension methods. So you have to <b>include</b> or <b>import</b> to use them.
</co>

<st>Definition</st>
<code-tab>
	<code lang="c#">
		DataReader ExecuteReader(this SQL.IQueryable o, 
								 Connection.Connection Connection, 
								 Connection.ExecuteIn ExecuteIn = Connection.ExecuteIn.ActualConnection)
	</code>
	<code lang="vb">
		ExecuteReader(o As SQL.IQueryable, 
					  Connection As Connection.Connection, 
					  Optional ExecuteIn As Connection.ExecuteIn = Connection.ExecuteIn.ActualConnection) As DataReader
	</code>
</code-tab>

<st>Examples</st>
<ew/>
<br/>
<br/>
<code-tab>
	<code lang="c#">
		using Odb; /* for extensions */

		/* ... */
		// to get the query result as DataReader. (SQL.IQueryable)
		var dt = sql.ExecuteReader(cn);
	</code>
	<code lang="vb">
		Imports Odb ' for extensions

		' ...

		' to get the query result as DataReader. (SQL.IQueryable)
		Dim dt = sql.ExecuteReader(cn, ForUpdate:=true)
	</code>
</code-tab>
