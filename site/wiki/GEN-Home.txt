<br/>

## What is it?
OrmDB is a simple, fast and lightweight embedded .NET ORM tool which was inspired by the equivalent other tools.
OrmDB works across .NET ADO providers as shown below.

|Type             | Driver                              |
|-----------------|-------------------------------------|
|Ms SQL           | System.Data.SqlClient               |
|Postgre SQL      | Npgsql                              |
|MySQL / MariaDB  | MySql.Data                          |
|Oracle           | ODP.Net (Oracle ManagedDataAccess)  |

<br/>

## Where to use?
* All desktop/web applications need to communicate with a database 
* Multi database operation
* Multi client application
* .Net / Asp.Net / .Net Core / Asp.Net Core
