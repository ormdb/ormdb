<k>CreateIndex</k> is used to create indexes in tables. Indexes are used to speed up searches/queries.

<br/>

><k>PRIMARY KEY cannot be created independent from a table, it can only be during table creation or deletion.</k>

<br/>

<st>Definition</st>
<code-tab>
	<code lang="c#">

			public class DDL {

				public string CreateIndex(Table.Base Table,
										  Table.IndexInfo IndexInfo,
										  string IndexName = null,
										  bool OnlyGenerateScript = false){}

				public string CreateIndex(string TableName,
										  Table.IndexInfo IndexInfo,
										  string IndexName = null,
										  bool OnlyGenerateScript = false){}

				public string CreateIndex(string TableName,
										  bool IsUnique,
										  IEnumerable<string> ColumnNames,
										  string IndexName = null,
										  bool OnlyGenerateScript = false){}
	</code>

<st>Examples</st>

<ew/>
<br/>

Following section demonstrates how to create index for a table.

<code-tab>
	<code lang="c#">
		// by table instance
		cn.DDL.CreateIndex(h, h.Indexes.First(), "my_index");

		// by table name
		cn.DDL.CreateIndex("table1", true, new[] { "column1", "column2" }, "my_index");
	</code>
	<code lang="vb">
		' by table instance
		cn.DDL.CreateIndex(h, h.Indexes.First(), "my_index")

		' by table name
		cn.DDL.CreateIndex("table1", True, { "column1", "column2" }, "my_index")
	</code>
</code-tab>

<br/>

Following section demonstrates how to generate <k>DROP INDEX</k> script.

<code-tab>
	<code lang="c#">
		var script = cn.DDL.CreateIndex(h, h.Indexes.First(), "my_index", true);
	</code>
	<code lang="vb">
		Dim script = cn.DDL.CreateIndex(h, h.Indexes.First(), "my_index", True)
	</code>
</code-tab>
