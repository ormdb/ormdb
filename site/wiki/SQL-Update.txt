**`Update`** method is used to update existing records in a table.

<br/>

**Definition**
``````XXX
```C#
namespace SQL {
    public class X {

        public static SQL.Update<T> Update<T>() 
                                    where T : Table.Base

        public static SQL.Update<T> Update<T>(Table.Field Where) 
                                    where T : Table.Base
```
```VB
NameSpace SQL
    Public Class X

		Public Shared Function Update(Of T As Table.Base)() 
									As SQL.Update(Of T)

		Public Shared Function Update(Of T As Table.Base)(Where As Table.Column) 
									As SQL.Update(Of T)
```
``````

<br/>

**Examples**

>Examples shown below needs the [Example Header](##OTH-Example-Header).

<br/>

Following section demonstrates **regular update**.

``````XXX
```C#
var sql = SQL.X.Update<TableHeader>(h.Id == 9999);

    var row = sql.Values;
        row.Note = "new note";

sql.Execute(cn);
```
```VB
Dim sql = SQL.X.Update(Of TableHeader)(h.Id = 9999)

    Dim row = sql.Values
        row.Note = "new note"

sql.Execute(cn)
```
``````

<br/>

Following section demonstrates **multiple update**.

``````XXX
```C#
var sql = SQL.X.Query;

sql.Execute(cn);
```
```VB
Dim sql = SQL.X.Query

``````

<br/>

Following section demonstrates **update with join**.

``````XXX
```C#
var sql = SQL.X.Update<TableHeader>;
    sql.InnerJoin(l, (l.Year == h.Year) & (l.Id == h.Id) );
    sql.Where(l.LineId > 3);

    var row = sql.Values;
        row.Note = "new note";

sql.Execute(cn);
```
```VB
Dim sql = SQL.X.Update(Of TableHeader)
    sql.InnerJoin(l, l.Year = h.Year And l.Id = h.Id )
    sql.Where(l.LineId > 3)

    Dim row = sql.Values
        row.Note = "new note"

sql.Execute(cn)
```
``````
>This sample updates the header records which contain DocumentLine that is greather than 3 in lines-table.
