**`Transaction`** methods are used with the [Connection](##CON-Create) object to save or cancel changes made to the data source.
Please refer to [Connection](##CON-Create) for further information.

<br/>

### BeginTrans
BeginTrans method starts a new transaction. A Transaction cannot be created , if there is an another transaction already in progress.

``````XXX
```C#
cn.BeginTrans();
```
```VB
cn.BeginTrans()
```
``````

<br/>

### CommitTrans
CommitTrans method saves all the changes made since the last BeginTrans method call and ends current transaction.

If a [Connection](##CON-Create) is opened as **`AutoTrans=true`** then each time **`CommitTrans`** method is invoked, a new transaction is created automatically.

``````XXX
```C#
cn.CommitTrans();
```
```VB
cn.CommitTrans()
```
``````

<br/>

If a [Connection](##CON-Create) is opened as **`AutoTrans=false`**, you can always start a new transaction as shown below.

``````XXX
```C#
cn.CommitTrans(AutoStartTrans:true);
```
```VB
cn.CommitTrans(AutoStartTrans:=true)
```
``````

<br/>

### RollbackTrans
RollbackTrans method cancels all changes made since the last BeginTrans method call and ends current transaction.

If a [Connection](##CON-Create) is opened as **`AutoTrans = true`** , then each time **`RollbackTrans`** method is invoked, a new transaction is created automatically.

``````XXX
```C#
cn.RollbackTrans();
```
```VB
cn.RollbackTrans()
```
``````

<br/>

If a [Connection](##CON-Create) is opened as **`AutoTrans = false`**, you can always start a new transaction as shown below.

``````XXX
```C#
cn.RollbackTrans(AutoStartTrans:true);
```
```VB
cn.RollbackTrans(AutoStartTrans:=true)
```
``````
