Obtains database [Table-Information] according to provider via [Database] currently connected.

<st>Definition</st>
<co type="light">
	<small><b>Namespace : </b>Odb.Connection.DDL</small>
</co>
<code-tab>
	<code lang="c#">
		public List<SchemaTableInfo> GetTables(string TableName = null)
	</code>
	<code lang="vb">
		Public Function GetTables(TableName As String = Nothing) As List(Of SchemaTableInfo)
	</code>
</code-tab>


<st>Returns</st>
<table>
	<thead>
		<th>SchemaTableInfo</th>
		<th>Description</th>
	</thead>
	<tr><td>Catalog</td><td>Database or Catalog name which has the table</td></tr>
	<tr><td>Schema </td><td>Name of schema that contains the table.     </td></tr>
	<tr><td>Table  </td><td>Table name.                                 </td></tr>
</table>

<st>Examples</st>
<ew/>
<br/>
<br/>

Following section demonstrates retrieving the tables from database.

<code-tab>
	<code lang="c#">
		var tables = cn.DDL.GetTables();

		var table_count = tables.Count;

		foreach (var table in tables) {
			System.Diagnostics.Debug.WriteLine($"{table.Catalog}.{table.Schema}.{table.Name}");
		}
	</code>
	<code lang="vb">
		Dim tables = cn.DDL.GetTables()

		Dim table_count = tables.Count

		For Each table In tables
			System.Diagnostics.Debug.WriteLine($"{table.Catalog}.{table.Schema}.{table.Name}")
		Next
	</code>
</code-tab>
