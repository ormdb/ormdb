Following section demonstrates using ExecuteCount clause. 
        var sql = SQL.X.Select()
                    .From(h)
                    .Where( h.Year.In(new[] {2018, 2019}) & 
                            h.Id.IsNotNull() );

        var count = sql.ExecuteCount(cn);
	
        Dim sql = SQL.X.Select() _
                    .From(h) _
                    .Where( h.Year.In({2018, 2019}) And 
                            h.Id.IsNotNull() )

        Dim count = sql.ExecuteCount(cn)


Also you can use COUNT clause, if you want. 
        var sql = SQL.X.Select(h.COUNT())
                    .From(h)
                    .Where( h.Year.In(new[] {2018, 2019}) & 
                            h.Id.IsNotNull() );

        var count = sql.Execute(cn);
	
        Dim sql = SQL.X.Select(h.COUNT()) _
                    .From(h) _
                    .Where( h.Year.In({2018, 2019}) And 
                            h.Id.IsNotNull() )

        Dim count = sql.Execute(cn)
