Examples

Following section demonstrates how to drop index of a table. 
		// by table instance
		cn.Provider.DropIndex(h, h.Indexes.First());

		// by table name
		cn.Provider.DropIndex("table1", "index_name");
	
		' by table instance
		cn.Provider.DropIndex(h, h.Indexes.First())

		' by table name
		cn.Provider.DropIndex("table1", "index_name")

Following section demonstrates how to generate DROP INDEX script. 
		var script = cn.Provider.DropIndex("table1", "index_name", true);
	
		Dim script = cn.Provider.DropIndex("table1", "index_name", True)
