Retrieves the info of the database product. This method may needs some privilege according to the database product and its version. Definition Namespace : Odb.Providers.IProvider 
		public ServerInfo GetServerInfo()
	
		Public Function GetServerInfo() As ServerInfo
Return
ServerInfo Description
Name Name of the database product.
Version Version of the database product.
Examples

Following section retrieves the product info from database server. 
		var si = cn.Provider.GetServerInfo();
		Debug.WriteLine($"Name={si.Name} , Version={si.Version}");
	
		Dim si = cn.Provider.GetServerInfo()
		Debug.WriteLine($"Name={si.Name} , Version={si.Version}")
