Following section demonstrates using SKIP and TOP clause. 
		// new instance of a table from definition
		var h = new TableHeader();
		
		// SQL is being initialized...
		var sql = SQL.X.Select().From(h)
					.Skip(5).Top(10)
					.OrderBy(h.Year);
		
		// Record(s) are being fetched
		var rows = sql.ExecuteReader(cn);
	
		' new instance of a table from definition
		Dim h = New TableHeader()
		
		' SQL is being initialized...
		Dim sql = SQL.X.Select().From(h) _
					.Skip(5).Top(10) _ 
					.OrderBy(h.Year)
		
		' Record(s) are being fetched
		Dim rows = sql.ExecuteReader(cn)
SKIP and TOP depends on the version of the database connected. For example, code shown above works only MsSql 2012 and above.
