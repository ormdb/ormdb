Tables (inherited from Odb.Table.Base) are used to be in sql queries. The tables defined can be used by its instances. Definition Namespace : Odb.Table Examples


Instances of the tables are being initialized. 
		// new instance of a table from definition
		var h = new TableHeader();
		var l = new TableLines();
		
		// execute a query
		var sql = SQL.X.Select().From(h);
	
		' new instance of a table from definition
		Dim h As New TableHeader()
		Dim l As New TableLines()
		
		' execute a query
		Dim sql = SQL.X.Select().From(h)


If you would like to join a table with itself, same table can be initialized twice as shown below. 
		// new instance of a table from definition
		var h  = new TableHeader();
		var l  = new TableLines();
		var h2 = new TableHeader();
		
		// execute a query
		var sql = SQL.X.Select(h.All, l.All, h2.Note.AS("note2"))
					   .From(h)
					   .LeftJoin(l, l.Id == h.Id )
					   .LeftJoin(l, h2.Id = h.Id );
	
		' new instance of a table from definition
		Dim h  As New TableHeader()
		Dim l  As New TableLines()
		Dim h2 As New TableHeader()
		
		' execute a query
		Dim sql = SQL.X.Select(h.All, l.All, h2.Note.AS("note2")) _
					   .From(h) _
					   .LeftJoin(l, l.Id = h.Id ) _
					   .LeftJoin(l, h2.Id = h.Id )
