OrmDb .Net
Release Date : 2020-03-19
Version : 1.5.0.2
Features
  • 1
    Code stability improvements, bug fixes and further improvements to performance.
  • 2
    Added LINQ support. ( Check Example )
  • 3
    Added DbCommand support. ( Check Example )
  • 4
    Added TSQL (as string) support. ( Check Example )
  • 5
    MinValue, MaxValue and DefaultValue attributes included in columns properties now. Please refer to Columns for further information.
  • 6
    Added new DDL functions such as AnalyzeTable, GetServerDateTime, GetServerInfo, PingServer.
  • 7
    Added new examples.
  • 8
    Wiki pages updated.
Breaking Changes
  • 1
    "Provider" classes removed. "Provider" classes are specified only on constructor of the Connection.
  • 2
    Connection object is not created through the provider. Connection is created by specifying the "provider" in the constructor.
  • 3
    "Execute" method removed. Instead of this, ExecuteReader and ExecuteNonQuery methods added.
  • 4
    "DataReader" removed. Instead of this, ExecuteReader returns the records as IEnumerable(Of Odb.DataRow).
  • 5
    "DataTable" removed.
  • 5
    "HasRow" removed. Instead of this, use code shown below. 

                            var sql = Select()...;
                        
                        var rows = sql.ExecuteReader(cn);
                        
                        var first_row = rows.First();
                        
                        var hasRow = (first_row != null);
	                
                            Dim sql = Select()...

                        Dim rows = sql.ExecuteReader(cn)

                        Dim first_row = rows.First()

                        Dim hasRow = (first_row IsNot Nothing)
If you experience a Breaking Change, please report the issue. Defects None
