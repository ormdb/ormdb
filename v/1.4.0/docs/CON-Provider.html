Provider is used to create the connection(s) which will be connect to a data source by supplying necessary information. Definition Namespace : Odb.Connection.Provider 
        Odb.Connection.Provider ( ProviderType Type, string Name, string ConnectionString )

        Odb.Connection.Provider ( ProviderType Type, IEnumerable<string> Name, string ConnectionString )
    
        Odb.Connection.Provider ( Type As ProviderType, Name As String, ConnectionString As String )

        Odb.Connection.Provider ( Type As ProviderType, Name As IEnumerable(Of String), ConnectionString As String )
Parameters
Parameters Description
Type is type of the DbProvider.
Name [1..n] is name of the DbFactory to be created. Can be specify as [InvariantName] or [AssemblyQualifiedName] or both.

[InvariantName] is the name of the factory in GAC; and the factory initialized from GAC by this name.

[AssemblyQualifiedName] is the name of the assembly from which provider object was loaded; and the factory initialized from references in memory of current process.

Examples:
InvariantName AssemblyQualifiedName
System.Data.SqlClientSystem.Data.SqlClient.SqlClientFactory, System.Data
MySql.Data.MySqlClientMySql.Data.MySqlClient.MySqlClientFactory, MySql.Data
Oracle.ManagedDataAccess.ClientOracle.ManagedDataAccess.Client.OracleClientFactory, Oracle.ManagedDataAccess
NpgsqlNpgsql.NpgsqlFactory, Npgsql
ConnectionString The connection string that includes the source database name, and other parameters needed to establish the initial connection.
Examples

Following section defines a provider. 
        // for MsSql
        var prv_ms = new Odb.Connection.Provider(ProviderType.MsSql, "System.Data.SqlClient", "Your-ConnectionString");

        // for PorstgreSql - multiple factory names
        var prv_pg = new Odb.Connection.Provider(ProviderType.PostgreSql, new[] {"Npgsql", "Npgsql.NpgsqlFactory, Npgsql"}, "Your-ConnectionString");
    
        ' for MsSql
        Dim prv_ms = new Odb.Connection.Provider(ProviderType.MsSql, "System.Data.SqlClient", "Your-ConnectionString")

        ' for PorstgreSql - multiple factory names
        Dim prv_pg = new Odb.Connection.Provider(ProviderType.PostgreSql, {"Npgsql", "Npgsql.NpgsqlFactory, Npgsql"}, "Your-ConnectionString")

You can create many Connection from the Provider. Please refer to Connection for further information.
