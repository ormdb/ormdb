Welcome to OrmDb documentation Wiki pages. Here you will find all information you need to know better OrmDb and how you can work.
This documentation is for latest version.
Download OrmDB has no dependencies. You can DOWNLOAD from releases, into your Bin folder and add it as Reference. What is it? OrmDB is a simple, fast and lightweight embedded .NET ORM tool which was inspired by the equivalent other tools. OrmDB works across .NET ADO providers as shown below.

Type Driver
System.Data.SqlClientSystem.Data.SqlClient.SqlClientFactory, System.Data
MySql.Data.MySqlClientMySql.Data.MySqlClient.MySqlClientFactory, MySql.Data
Oracle.ManagedDataAccess.ClientOracle.ManagedDataAccess.Client.OracleClientFactory, Oracle.ManagedDataAccess
NpgsqlNpgsql.NpgsqlFactory, Npgsql
Where to use?
  • All desktop/web applications need to communicate with a database
  • Multi database operation
  • Multi client application
  • .Net / Asp.Net / .Net Core / Asp.Net Core
