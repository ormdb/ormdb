Executes the AnalyzeTable on the database server. This method generates table statistics. Examples Following section demonstrates ANALYZE the table(s). 
		var t0 = new Orders();

        // by instance
        connection.Provider.AnalyzeTable(t0);

        // by name
        connection.Provider.AnalyzeTable("table1");

The parameter OnlyGenerateScript should be true, if the command's script is requested. Default value of this parameter is false. 
        var script = connection.Provider.AnalyzeTable(..., OnlyGenerateScript:true);
