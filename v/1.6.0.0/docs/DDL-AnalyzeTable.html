Executes the AnalyzeTable on the database server. This method generates table statistics. Examples Following section demonstrates ANALYZE the table(s). 
        // by instance
        cn.Provider.AnalyzeTable(h);

        // by name
        cn.Provider.AnalyzeTable("table1");

Following section demonstrates how to generate ANALYZE script. 
        var script = cn.Provider.AnalyzeTable("table1", true);
