TRUNCATE method is used to delete existing records in a table like DELETE. This process also executes the UPDATE STATISTICS for tables, after DELETE operation. This method may needs some privilege according to the database product and its version. Examples Following section demonstrates TRUNCATE the table(s). 
        // get provider for the connection
		var cp = cn.Provider();

        // by instance
        var t0 = new Orders().AsQueryable();
        var t1 = new OrderDetails().AsQueryable();
        cp.TruncateTable(t0);   // single
        cp.TruncateTable(new [] {t0, t1});  // multi

        // by type
        cp.TruncateTable<Orders>();

        // single table by name
        cp.TruncateTable("table1");
