TRUNCATE method is used to delete existing records in a table like DELETE. This process also executes the UPDATE STATISTICS for tables, after DELETE operation. Examples Following section demonstrates TRUNCATE the table(s). 
        var t0 = new Orders();
        var t1 = new OrderDetails();

        // single table by instance
        connection.Provider.TruncateTable(t0);

        // multi table by instance
        connection.Provider.TruncateTable(new [] {t0, t1});

        // by type
        connection.Provider.TruncateTable<Orders>();

        // single table by name
        connection.Provider.TruncateTable("table1");
for all databases
TRUNCATE TABLE [Orders]


The parameter OnlyGenerateScript should be true, if the command's script is requested. Default value of this parameter is false. 
        var script = cn.Provider.TruncateTable(..., OnlyGenerateScript:true);
