DropTable is used to drop an existing table in a database. Examples Following section demonstrates DROP the table(s). 
		var t0 = new Orders();
        var t1 = new OrderDetails();

		// single table by instance
		connection.Provider.DropTable(t0);

		// multi table by instance
		connection.Provider.DropTable(new [] {t0, t1});

        // by type
        connection.Provider.DropTable<Orders>();

		// single table by name
		connection.Provider.DropTable("table1");

The parameter OnlyGenerateScript should be true, if the command's script is requested. Default value of this parameter is false. 
        var script = cn.Provider.DropTable(..., OnlyGenerateScript:true);
