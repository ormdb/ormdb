DropTable is used to drop an existing table in a database. Examples Following section demonstrates how to drop table. // single table by instance cn.Provider.DropTable(h); // multi table by instance cn.Provider.DropTable(new [] {h, l}); // single table by name cn.Provider.DropTable("table1");
Following section demonstrates how to generate DROP TABLE script. var script = cn.Provider.DropTable(h, true);
