Tables (inherited from Odb.Table.Base) are used to be in sql queries. The tables defined can be used by its instances. Definition Namespace : Odb.Table Examples 

		var th = new TableHeader();

		var sql = Odb.SQL.X.Select(x => x.From(th));
	
		Dim th As New TableHeader()

		Dim sql = Odb.SQL.X.Select(Function(X) X.From(th))
SELECT *
FROM header AS th

If you would like to join a table with itself, same table can be initialized twice as shown below. 
		var t1 = new TableHeader();
		var t2 = new TableHeader();

		var sql = SQL.Select(x => x.From(t1)
								   .LeftJoin(t2, t2.Id == t1.Id));
	
		Dim t1 = new TableHeader()
		Dim t2 = new TableHeader()
		
		Dim sql = SQL.Select(Function(X) X.From(t1) _
								          .LeftJoin(t2, t2.Id = t1.Id))
SELECT *
FROM header AS t1
LEFT JOIN header AS t2 ON t2.id = t1.id
