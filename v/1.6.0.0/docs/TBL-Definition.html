Tables (inherited from Odb.Table.Base) are used to be in sql queries. The tables defined can be used by its instances. Examples 

        var t0 = new Orders();

        Odb.SQL.X.Select(x => x.From(t0));
SELECT *
FROM [Orders] AS t0 

        var t0 = new Orders();
        var t1 = new OrderDetails();

        SQL.Select(x => x.From(t0)
                         .LeftJoin(t1, t1.OrderId == t0.OrderId)
        );
SELECT *
FROM [Orders] AS t0
LEFT JOIN [OrderDetails] AS t1 ON t1.OrderId = t0.OrderId
