CreateTable function is used to create a new table in a database. This method may needs some privilege according to the database product and its version.

SchemaPrefix is a prefix for table-name. (Exm. Name="table-name" , NameWithPrefix="[pfx]table-name"). Examples Following section demonstrates creating a table. 
        // get provider for the connection
		var cp = cn.Provider();

		// by table instance
		var t0 = new Orders().AsQueryable();
        cp.CreateTable(t0);
