CreateTable function is used to create a new table in a database.
SchemaPrefix is a prefix for table-name. (Exm. Name="table-name" , NameWithPrefix="[pfx]table-name"). Examples Following section demonstrates creating a table. 
        // create the table.
        cn.Provider.CreateTable(h);

Following section demonstrates how to generate CREATE TABLE script. 
        var script1 = cn.Provider.CreateTable(h, OnlyGenerateScript:true);
