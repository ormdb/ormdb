CreateIndex is used to create indexes in tables. Indexes are used to speed up searches/queries. PRIMARY KEY cannot be created independent from a table, it can only created during table creation or deletion. Examples Following section demonstrates how to create index for a table. 
		var t0 = new Orders();

		// by table instance
		connection.Provider.CreateIndex(t0, t0.Indexes.First(), "my_index");

		// by table name
		connection.Provider.CreateIndex("table1", true, new[] { "column1", "column2" }, "my_index");
