CreateIndex is used to create indexes in tables. Indexes are used to speed up searches/queries. PRIMARY KEY cannot be created independent from a table, it can only created during table creation or deletion. Examples Following section demonstrates how to create index for a table. // by table instance cn.Provider.CreateIndex(h, h.Indexes.First(), "my_index"); // by table name cn.Provider.CreateIndex("table1", true, new[] { "column1", "column2" }, "my_index");
Following section demonstrates how to generate CREATE INDEX script. var script = cn.Provider.CreateIndex(h, h.Indexes.First(), "my_index", true);
