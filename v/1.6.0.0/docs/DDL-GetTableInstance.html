GetTableInstance is used to initialize the instance of [Table.Base.Generic], according to table-information in database. Examples Following section demonstrates generating the instance of the table. 
		var tb0 = new Orders();

		// via provider
		var tb1 = connection.Provider.GetTableInstance<TableHeader>();
		var tb2 = connection.Provider.GetTableInstance(typeof(TableHeader)());
		var tb3 = connection.Provider.GetTableInstance("TableHeader");
