GetTableInstance is used to initialize the instance of [Table.Base.Generic], according to table-information in database. Examples Following section demonstrates generating the instance of the table. 
		// instance(s) created from definition(s)
		var t1 = new TableHeader();

		// new instance of a table via provider
		var tb1 = cn.Provider.GetTableInstance<TableHeader>();
		var tb2 = cn.Provider.GetTableInstance(typeof(TableHeader)());
		var tb3 = cn.Provider.GetTableInstance("TableHeader");

Following section demonstrates reading the informations from table-instance. 
		// Reading table columns...
		foreach ( var col in tb.GetColumns() ){

		// Reading table indexes...
