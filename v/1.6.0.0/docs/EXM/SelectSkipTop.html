Following section demonstrates using SKIP and TOP clause. 
        var th = new TableHeader();

        var sql = Odb.SQL.Select(x => 
                        x.From(th)
                         .Skip(15).Top(5)
                    );
	
        Dim th = new TableHeader()

        Dim sql = Odb.SQL.Select(Function(x) _
                        x.From(th) _
                         .Skip(15).Top(5)
                    )
SKIP and TOP depends on the version of the database connected. For example, code shown above works only MsSql 2012 and above.
