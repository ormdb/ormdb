                var sql = Odb.SQL.Insert();

                var dt = DateTime.Now;
                sql.Add<TableHeader>(dt.Year, 1, dt, null, DBNull.Value);

                sql.Add<TableLines>(dt.Year, 1, 1, "note 1");
                sql.Add<TableLines>(dt.Year, 1, 2, "note 2");

                var affectedRows = sql.ExecuteNonQuery(cn);
In this sample; column "IsActive" was not specified (3rd value), and also column "Note" was specified as DbNull (4th value).
These columns will be accepted as its initial value, since the parameter "SetInitials" of Insert is TRUE.
