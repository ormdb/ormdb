Following section demonstrates insertion with Value Array. 
        var sql = Odb.SQL.Insert(true);

        sql.Add<TableHeader>(DateTime.Now.Year, 
                            1, 
                            DateTime.Now, 
                            null, 
                            DBNull.Value);
	
        Dim sql = Odb.SQL.Insert(True)

        sql.Add(Of TableHeader)(DateTime.Now.Year,
                                1,
                                DateTime.Now,
                                Nothing,
                                DBNull.Value)
In this sample; column "IsActive" was not specified (3rd value), and also column "Note" was specified as DbNull (4th value).
These columns will be accepted as its initial value, since the parameter "SetInitials" of Insert is TRUE.
