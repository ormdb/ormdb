Following section demonstrates how to delete from [TableHeader] records, which contain records that is between 1-10 in [TableLines]. 
		var th = new TableHeader();
		var tl = new TableLines();
		
		var sql = Odb.SQL.Delete(x =>
						x.From(th) 
						 .InnerJoin(tl,
									tl.Year == th.Year &
									tl.Id == th.Id) 
						 .Where(tl.LineId > 1M &
								tl.LineId < 10M)
					);
	
		Dim th = new TableHeader()
		Dim tl = new TableLines()
		
		Dim sql = Odb.SQL.Delete(Function(x) _
						x.From(th) _
						 .InnerJoin(tl,
									tl.Year = th.Year And
									tl.Id = th.Id) _
						 .Where(tl.LineId > 1D And
								tl.LineId < 10D)
					)
This query generates different results for each database connected.
