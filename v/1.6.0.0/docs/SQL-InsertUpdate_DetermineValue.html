Rules are shown below for determining the value for updating or inserting processes.
Value Rule
Null/Nothing If the value is "Null or Nothing" then Column Default Value is accepted.
If the Column Default Value is not specified then it is accepted initial value of data type.
{
ColumnString : String.Empty
ColumnNumeric : 0 (Zero)
ColumnBoolean : false
ColumnBinary : Array.Empty
ColumnTime : TimeSpan.Zero
ColumnDate : Date.MinValue.Date
ColumnDateTime : Date.MinValue
}
DbNull If NULL value will be used especially on insert or update processes; then DbNull value has to be specified.
Other All values except "Null/Nothing" and "DbNull". Value is accepted as is.
