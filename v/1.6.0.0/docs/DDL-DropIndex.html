DropIndex is used to drop an existing index of a table in a database. Examples Following section demonstrates DROP-INDEX of the table(s). 
		var t0 = new Orders();

		// by table instance
		connection.Provider.DropIndex(t0, t0.Indexes.First());

        // by type
		connection.Provider.DropIndex<Orders>("index_name");

		// by table name
		connection.Provider.DropIndex("table1", "index_name");
MsSql
DROP INDEX IF EXISTS [Index] ON [Table]

MySql
DROP INDEX [Index] ON [Table]

PostgreSql
DROP INDEX IF EXISTS [Index]

Oracle
DROP INDEX [Index]
The parameter OnlyGenerateScript should be true, if the command's script is requested. Default value of this parameter is false. 
        var script = cn.Provider.DropIndex(..., OnlyGenerateScript:true);
